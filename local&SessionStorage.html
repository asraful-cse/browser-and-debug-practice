27-8.. local storage and session storage ,edit storage information


The read-only sessionStorage property accesses a session Storage object for the current origin. sessionStorage is similar to localStorage; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.
কেবলমাত্র পঠন সেশন স্টোরেজ সম্পত্তি বর্তমান উত্সের জন্য একটি সেশন স্টোরেজ অবজেক্টে অ্যাক্সেস করে। সেশন স্টোরেজ লোকালস্টোরেশনের সমান; পার্থক্যটি হ'ল লোকালস্টোরারে ডেটার মেয়াদ শেষ না হওয়ার পরে, পৃষ্ঠা সেশন শেষ হওয়ার পরে সেশনসটোরেজে ডেটা সাফ হয়ে যায়।
